Microbakery Database

A database management system for all your small-scale or home bakery needs!

About the site

GitHub Repo

This lets the baker add, delete, and select records to keep track of customers, orders, recipes, and more. Tables include: Customer, Order, Product, LineItem, Recipe, Ingredient, Flour. This app was created with Flask and uses a relational PostgreSQL database hoosted on Heroku. Bulma is used for CSS styling.

Add to a table!

Go to route /add_[table name] to go to the form to add to that table!

Show a table!

Go to route /show_[table name] to see all records in that table!

Select from a table!

Go to route /select_[table name] to select a record from that table!

Delete from a table!

Go to route /delete_[table name] to delete a record from table!
